When the age varies, we don't know how does the computer time vary.

t_test.py and permutation.py

The t_test statistics is 1.77696715175 and the p_value is 0.075721462062 which is larger than 0.05, the significance level. So we accept the null hypothesis that there no difference. However, the p value is low which means either that the null hypothesis is true and a highly improbable event has occurred, or that the null hypothesis is false. So there is no enough evidence to suggest that the average number of hours spent on computer per week of males is equal to that of females.

In the permutation test, we randomly pick permutation 1000 times.Here is one experiment's result:

  • The probability of observing the mean 0.890522973787 when assume the null hypothesis is true: 0.068.
Since the significance level is 0.05, we accept the null hypothesis. But again, this only means either that the null hypothesis is true and a highly improbable event has occurred, or that the null hypothesis is false. So we get the same conclusion as in t-test.
